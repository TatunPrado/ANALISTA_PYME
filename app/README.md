# APP Layer

## Propósito

La carpeta APP contiene la capa SaaS de Klar Analytics.

No reemplaza:

* Agents
* Skills
* Knowledge
* Playbooks
* Operations

La APP consume esos componentes mediante Knowledge Engine y los transforma en una experiencia interactiva para el usuario final.

## Componentes

### Session

Gestión del ciclo de vida del diagnóstico.

### Objectives

Mapeo entre objetivos empresariales y dimensiones de análisis.

### Findings

Clasificación de hallazgos.

### Scoring

Evaluación de madurez.

### Dashboard

Visualización en tiempo real.

### Reports

Generación de informes ejecutivos.

# Dashboard Definition

## Objetivo

Mostrar el progreso del diagnóstico en tiempo real.

## Componentes

### Progreso

* porcentaje completado

### Hallazgos

* fortalezas
* debilidades
* riesgos
* oportunidades

### Madurez

* score general
* score por dimensión

### Confianza

* nivel de confianza del diagnóstico

## Actualización

Cada interacción del usuario debe actualizar:

* progreso
* confianza
* hallazgos
* scores

## 1. Concepto Visual Principal

**Precisión que revela claridad.**

El logotipo elegido es el concepto **Señal / Indicador**: una gráfica con una línea de base estable que de repente marca un pico preciso, rematado por un punto sólido. Es la representación visual del acto de diagnosticar: detectar la anomalía, encontrar lo que se desvía de la norma, señalar con exactitud dónde está el hallazgo.

El símbolo comunica cuatro ideas en un solo trazo:
1. **Línea de base** → la empresa en su estado actual
2. **El pico** → el hallazgo, la anomalía detectada
3. **El punto en la cima** → la precisión del diagnóstico
4. **El punto teal interior** → la claridad que aportamos

El concepto central no es "inteligencia artificial" ni "tecnología". El concepto central es **el diagnóstico**: la capacidad de mirar, comprender y señalar con precisión lo que otros no ven.

### Traducción visual

| Elemento conceptual | Elemento visual en el logo |
|---|---|
| Examinar | Línea de base con datos (puntos en la gráfica) |
| Revelar | El pico que se eleva abruptamente |
| Precisión | Punto sólido exacto en la cima del pico |
| Claridad | Punto teal interior en el centro del marcador |
| Decisión | Dirección clara post-pico |

---

## 2. Concepto Visual Secundario

**La aguja que marca el hallazgo.**

En su versión mínima, el logotipo se reduce al pico con el punto: una "V" invertida con un marcador en el vértice. Esto funciona como favicon, avatar de redes y marca de agua.

### Relación entre conceptos

```
Concepto Principal (línea de base + pico + punto)
      ↓ simplificación
Concepto Secundario (pico + punto — solo el hallazgo)
      ↓ abstracción
Marca de agua / Icono (16–32px) — el punto con un destello
```

## 3. Metáforas Visuales Posibles

### Metáfora elegida — La señal / El indicador

Una línea de base con un pico preciso que se destaca. Un punto sólido marca la cima. Representa la detección de un hallazgo, la anomalía que merece atención, el momento en que los datos revelan una verdad.

**Ventaja**: Conexión directa e intuitiva con el mundo del diagnóstico. No necesita explicación.

### Metáfora 1 — La cuadrícula de análisis (alternativa evaluada)

Una retícula o cuadrícula parcial donde un sector está "iluminado", "marcado" o "revelado". Representa el proceso de poner orden, medir, diagnosticar.

**Ventaja**: Única en el mercado de consultoría.

### Metáfora 2 — El prisma (alternativa evaluada)

Un haz de luz que entra y se descompone en bandas de color o datos.

### Metáfora 3 — La diana / El punto focal (alternativa evaluada)

Un blanco o diana con un punto en el centro. Representa precisión y enfoque.

### Metáfora 4 — Las capas que se revelan (alternativa evaluada)

Formas superpuestas donde una capa superior se corre o se abre para revelar una capa inferior.

## 6. Arquitectura del Sistema de Logotipo

### Estructura de variantes

```
                    LOGOTIPO PRINCIPAL
                    ┌──────────┐
                    │  Símbolo │
                    │  +       │
                    │  Nombre  │
                    └──────────┘
                          │
          ┌───────────────┼───────────────┐
          │               │               │
    HORIZONTAL       VERTICAL       SÍMBOLO
    Símbolo +        Símbolo          Solo
    Nombre a         + Nombre     (avatar, favicon,
    la derecha       abajo         app icon)
          │               │               │
          ├───────┐       ├───────┐       │
      Color     Mono   Color     Mono     │
      Full      (B/N)  Full      (B/N)    │
          │               │               │
          └───────┬───────┘               │
                  │                       │
            LOGOTIPO                  SÍMBOLO
            COMPACTO                  MONO
          (responsive)            (marca de agua)
```

## 7. Versiones Horizontales

### Uso: Header de web, firma de email, papelería, presentaciones

```
─────────────────────────────────────────────
                  400px (ej.)
┌──────┐  ┌─────────────────────────────────┐
│      │  │  NOMBRE DE LA CONSULTORA         │
│ SÍM- │  │  Diagnóstico empresarial con IA  │
│ BOLO │  └─────────────────────────────────┘
└──────┘
```

| Elemento | Especificación |
|---|---|
| Relación símbolo/isotipo | Símbolo: 25–30% del ancho total |
| Espaciado entre símbolo y texto | 2× el ancho del trazo del símbolo |
| Tagline | Sí, debajo del nombre en versión horizontal extendida |
| Altura máxima | 48–56px para web, 32px para email signature |
| Versión responsive | A partir de 320px de ancho, pasar a símbolo + solo nombre |

### Breakpoints horizontales

| Ancho disponible | Versión |
|---|---|
| ≥ 400px | Horizontal con tagline |
| 280–399px | Horizontal sin tagline |
| 180–279px | Símbolo + inicial (o abreviatura) |
| < 180px | Solo símbolo |
